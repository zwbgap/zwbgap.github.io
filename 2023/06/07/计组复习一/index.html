计算机系统概论

1. 计算机系统的组成

计算机系统由硬件和软件两部分组成。

1.1 硬件组成

计算机硬件包括以下几个主要部分：

  • 中央处理器（Central Processing Unit，CPU）：负责执行指令、控制和处理数据。
  • 主存储器（Main Memory）：用于存储指令和数据。
  • 输入/输出设备（Input/Output Devices）：用于与外部世界进行交互，如键盘、显示器、磁盘等。
  • 系统总线（System Bus）：用于各个硬件组件之间的数据传输。

1.2 软件组成

计算机软件包括以下两类：

  • 系统软件：控制和管理计算机硬件资源，提供基本的功能和服务，如操作系统、编译器、驱动程序等。
  • 应用软件：用于解决特定的问题或满足用户的需求，如办公软件、数据库管理系统、图像处理软件等。

2. 计算机的层次结构

计算机的层次结构由下至上分为以下几层：

  1. 物理层：包括计算机的硬件电路，例如门电路、寄存器、ALU等。
  2. 数字逻辑层：利用门电路和触发器等构建组合逻辑和时序逻辑电路，如加法器、多路选择器等。
  3. 微程序层：利用微指令控制硬件电路的运行，实现指令的执行和控制。
  4. 指令系统层：定义计算机的指令集和指令格式，如MIPS指令集、x86指令集等。
  5. 操作系统层：控制和管理计算机的硬件资源，提供基本的功能和服务。
  6. 高级语言层：使用高级语言编写程序，通过编译器或解释器转换为机器语言。
  7. 应用程序层：用于解决特定的问题或满足用户的需求。

3. 计算机的工作原理

计算机的工作原理可以简单概括为以下几个步骤：

  1. 取指令（Instruction Fetch，IF）：从主存储器中读取指令，并将指令存入指令寄存器（IR）。
  2. 指令译码（Instruction Decode，ID）：对指令进行译码，确定指令的操作类型和操作数。
  3. 执行指令（Execute，EX）：根据指令的操作类型和操作数，执行相应的操作。
  4. 访存取数（Memory Access，MA）：如果指令需要访问主存储器，则进行存储器读写操作。
  5. 写回结果（Write Back，WB）：将执行结果写回寄存器或存储器。

4. 计算机的性能指标

衡量计算机性能的主要指标包括以下几个：

  • 时钟周期（Clock Cycle）：指一条指令所需的时钟脉冲数。
  • 时钟频率（Clock Frequency）：时钟信号的频率，单位为赫兹（Hz）。
  • CPI（Cycles Per Instruction）：指令的平均时钟周期数。
  • 执行时间（Execution Time）：完成一段程序所需的时间。
  • 吞吐量（Throughput）：单位时间内完成的任务数。

5. 计算机发展的历程

计算机发展经历了以下几个阶段：

  • 电子管时代：使用电子管作为主要的电子元件，如第一台电子管计算机 ENIAC。
  • 晶体管时代：晶体管取代了电子管，计算机变得更小、更可靠、更高效。
  • 集成电路时代：集成电路（IC）的出现使得多个晶体管集成在一个芯片上，计算机性能大幅提升。
  • 超大规模集成电路时代：超大规模集成电路（VLSI）技术使得更多的晶体管集成在一个芯片上，计算机进一步提升性能。
  • 并行计算时代：利用多个处理器同时执行任务，提高计算机的并行处理能力。
  • 网络时代：计算机之间通过网络连接，实现信息的共享和通信。
